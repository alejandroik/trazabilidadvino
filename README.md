## Instructivo

A continuación se detallan los pasos necesarios para hacer funcionar el prototipo de manera local.

### Pre-requisitos

Es necesario instalar los siguientes componentes para configurar el entorno.
* [Tomcat 8](https://tomcat.apache.org/download-80.cgi)
* [MySQL Community Server 5.6.50](https://dev.mysql.com/downloads/mysql/5.6.html)
* [Java SDK 8u271](https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html)
* [Ganache](https://www.trufflesuite.com/ganache)
* [Git](https://git-scm.com/downloads)


### Instalación

1. Clonar el repositorio
```sh
git clone https://github.com/alejandroik/trazabilidadvino.git
```
2. Copiar el archivo TrazaVinoWeb.war al directorio webapps de tomcat
